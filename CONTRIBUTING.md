# Contributing to seethru.media

## Getting Started

1. Fork the repo
2. Clone your fork
3. Run `yarn install`
4. Run `yarn dev`

## Structure

```
apps/         - Next.js applications
packages/     - Shared packages
docs/         - Governance & policy docs
scripts/      - Automation
```

## Pull Request Process

1. Create a feature branch
2. Make your changes
3. Run `yarn lint` and `yarn typecheck`
4. Submit PR with clear description

## Governance

All governance documents are in `docs/`. Changes follow the Amendment Process.

## Code of Conduct

Be respectful. Assume good faith. Disagree constructively.
