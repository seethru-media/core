# Amendment Process

## How Governance Changes

### 1. Proposal
Anyone can propose a governance change via GitHub Issue in this repo.

### 2. Comment Period
14-day public comment period. All comments are public.

### 3. Review
Founder + Editorial Board (when established) review comments.

### 4. Community Poll (Major Changes)
For significant changes (new red lines, policy reversals):
- Community poll (advisory, not binding initially)
- Binding when Community Appeals Board is established (Phase 3)

### 5. Decision
Published with rationale explaining how comments were considered.

## What Can Be Changed

- All governance documents
- Moderation guidelines
- Juror training materials
- Policy thresholds

## What Cannot Be Changed

- Core trust principles (without supermajority)
- Retroactive policy application
- Removing forkability right
